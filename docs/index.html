Features

Everything you need to get your notes off the tablet and into your workflow.

USB & Wi-Fi Sync — connect via cable or wirelessly over your local network
Incremental Backup — only downloads files that have changed
PDF Conversion — full v5 and v6 .rm support with template backgrounds
AI Handwriting Recognition — transcribe handwriting with GitHub Copilot, Claude, or Google Gemini
Markdown Export — each notebook page becomes a .md file with frontmatter and images
Watch Mode — automatic periodic sync with a system-tray icon

Device Compatibility

Current device support and the caveats to know before you set things up.

  • reMarkable 2 — Known to work. The primary device used for testing.
  • reMarkable Paper Pro — Known to work with setup caveat. SSH must be enabled through developer mode, which currently requires a factory reset.
  • reMarkable 1 — Not currently verified. Compatibility is not guaranteed until tested directly.

Getting Started

1

Configure

Run the setup wizard to connect to your tablet and choose an AI provider (GitHub Copilot, Claude, or Google Gemini).

reMarkableSync config
2

Watch

Start watch mode to sync automatically on a schedule. A system tray icon lets you pause, sync now, change the interval, and open output folders.

reMarkableSync watch
3

Run at Startup

Want it always running? Right-click the system tray icon and enable "Run at Startup" — reMarkableSync will launch automatically when you log in.
